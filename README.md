# 指纹识别传感器 Mind+ 用户库

![](./arduinoC/_images/featured.png)

## 简介

本扩展库为指纹识别传感器设计，支持录入指纹与识别指纹等功能。

支持 Mind+ 导入库，**要求 Mind+ 软件版本为 1.6.2 及以上**。
